Véhiparc

Véhiparc est une application Web permettant la gestion de flotte de véhicules d'entreprise. Le développement est réalisé par le CSMSI - LA POSTE, la maîtrise d'ouvrage est une branche de La Poste: Vehiposte.

Ma mission sur ce projet a été de changer le design du site en suivant une charte graphique ainsi que de réaliser un système d'aide pour mieux comprendre certaines pages qui peuvent être complexes. Ce dernier aura une partie administration qui permettra la gestion des aides.

Ce projet est réalisé avec le framework Angular,
Trois bases de données : MongoDB, SQL, Oracle.
Une WebApp en Node.js qui accède à la base de données SQL qui utilise l'ORM Bookshelf.js.
Un backend qui accède à la base de données MongoDB en Node.js également.
Le versionning est géré par GitLab.



Application Mobile

Le projet est une application mobile permettant la gestion des rendez-vous pour les infirmiers à domicile. Ce projet a été réalisé dans le cadre de PPE (Projet Personnel Encadré) au sein de l'établissement scolaire Lycée Carcouët
Ce projet est réalisé avec le langage JAVA.





Vidéothèque

Le projet est une application web permettant la gestion d'emprunt de films. Ce projet a été réalisé dans le cadre de PPE (Projet Personnel Encadré) au sein de l'établissement scolaire Lycée Carcouët.


Ce projet est réalisé avec le langage PHP, HTML/CSS, Ajax via JQuery.
Base de données : MySQL.
Le versionning est géré par Github.
