Véhiparc

Véhiparc est une application Web permettant la gestion de flotte de véhicule d'entreprise. Le developpement est réalisé par le CSMSI - LA POST, la maitrise d'ouvrage est une branche de La Poste -> Vehiposte.
Ce projet est réaliser avec le framework Angular.
Le versionning est gérer par GitLab.
