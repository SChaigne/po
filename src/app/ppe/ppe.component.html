Véhiparc

Véhiparc est une application Web permettant la gestion de flotte de véhicules d'entreprise. Le développement est réalisé par le CSMSI - LA POST, la maîtrise d'ouvrage est une branche de La Poste: Vehiposte.
Ma mission sur ce projet a été de changer le design du site en suivant une charte graphique ainsi que de réaliser un système d'aide pour mieux comprendre certaines pages qui peuvent être complexe. Ce dernier aura une partie administration qui permettra la gestion des aides.
Ce projet est réaliser avec le framework Angular.
Le versionning est gérer par GitLab.

Application Mobile

Application mobile est une application Mobile permettant la gestion des rendez-vous pour les infirmier à domicile. Ce projet à été réaliser dans le cadre de PPE (Projet Personnel Encadré) au sein de l'établissement scolaire "Lycée Carcouët"
Ce projet est réaliser avec le framework Flutter qui utilise le langage Dart.
